# Trabalho Final de Programação Web I
Repositório destinado ao projeto final da disciplina de Programação Web I.

-> Hotel Flash - Site de Reservas

Este é um projeto de um site de reservas de hotéis desenvolvido com HTML, CSS e JavaScript. O objetivo é simular uma plataforma simples onde o usuário pode visualizar informações de um determinado hotel, fazer e buscar reservas, e acessar seções como avaliações e sobre.

---

## Funcionalidades

- Página inicial com informações do serviço Hotel Flash (seção "Sobre");
- Navegação com menu interativo;
- Seção "Hotéis" com apresentação dos hotéis "filiados";
- Seção "Reservas" com sistema de reserva (interface);
- Seção de "Avaliações" com avaliações de clientes sobre os hotéis;
- Seção "Atrativos" falando um pouco mais sobre cada cidade dos hotéis cadastrados;
- Seção "Suporte" com as formas de contato com o suporte do Hotel Flash;
- Animações e efeitos com JavaScript;
- Scroll automático entre seções.

---

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- PostgreSQL
